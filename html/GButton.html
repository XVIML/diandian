Gradient Button Widget

Gradient Button Widget is a imporvement of button widget. It can receive user's mouse input and turn it into a event stream. By bind the event stream to a callback function or another widget, user can let the program do a designed work. Below is a Grident Button in GUI.

You can insert a gradient button widget in the GUI by:

hl=dr.DRGButton=(win,x,y,w,h,bg1,bg2,fg,R,Title,fSize,ID)

where hl is the handle of returned Button widget. win is the handle of GUI, (x,y,w,h) is the insert position, (bg1,bg2,fg) is the background color and foreground color, fsize is the fontsize of title, and ID is the key value. Below is a sample codes.

mGBut1=dr.DRGButton(win,68,30,100,40,'#00ccaa','#eeee00','#000000',12,'GButton1',12,101)

You can bind a callback function or a set method of another widget to the gradient button's event stream, by£º

hl.addCallBackSingle(setValueSingle)

The setValueSingle function is defined as:

def setValueSingle(v):

When the gradient button is pressed, the binded callback function will receive the key value of the gradient button.

Gradient Button Widget is a wrap of Tkinter Canvas widget, so you can set it by Tkinter Canvas's method.
