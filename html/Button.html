Button Widget

Button is most commonly used GUI widget. It can receive user's mouse input and turn it into a event stream. By bind the event stream to a callback function or another widget, user can let the program do a designed work. Below is a Button in GUI.

You can insert a Button widget in the GUI by:

hl=dr.DRButtonX(win,x,y,w,h,bg,fg,title,fsize,ID)

where hl is the handle of returned Button widget. win is the handle of GUI, (x,y,w,h) is the insert position, (bg,fg) is the background color and foreground color, fsize is the fontsize of title, and ID is the key value. Below is a sample codes.

mBut1=dr.DRButtonX(win,127,30,100,40,'#00aacc','#eeee00','Button1',12,101)

You can bind a callback function or a set method of another widget to the button's event stream, by

hl.addCallBackSingle(setValueSingle)

The setValueSingle function is defined as:

def setValueSingle(v):

When the button is pressed, the binded callback function will receive the key value of the button.

Button Widget is a wrap of Tkinter Button widget, so you can set it by Tkinter Button's method. For example, you can change the border of the button to 10 pixels.

hl.ax.config(bd=10)

For Tkinter Button, you can use configure function to change its looklike. These setting include:

text='Title'
bg='#Background color code'
fg='#Text color code'
activebackground='#active backgrounf color code'
activeforeground='#active text color code'
relief='Border Style name'(flat、sunken、raised、groove、ridge、solid)
bd=Border Pixels
font=('Name',pixels)
